# Quick Reference

> One-line summary of all concepts. See [Concepts Dictionary](concepts-dictionary.md) for details.

## React Core

- **useState**: Manage mutable state within components → [Details](concepts-dictionary.md#usestate)
- **useEffect**: Handle component lifecycle and side effects → [Details](concepts-dictionary.md#useeffect)
- **useRef**: Persist values across renders and access DOM directly → [Details](concepts-dictionary.md#useref)
- **useReducer**: Manage complex state logic with reducer functions → [Details](concepts-dictionary.md#usereducer)
- **useCallback**: Memoize functions to prevent unnecessary recreation → [Details](concepts-dictionary.md#usecallback)
- **Custom Hooks**: Extract reusable logic into Hooks → [Details](concepts-dictionary.md#custom-hooks)
- **Props**: Pass data from parent to child components → [Details](concepts-dictionary.md#props)
